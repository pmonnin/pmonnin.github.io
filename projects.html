Software & Data

  • PGxLOD, a knowledge graph integrating pharmacogenomic knowledge from various sources
  • PGxO, an ontology to represent pharmacogenomic knowledge
  • YAGO4-LP, link prediction datasets based on the YAGO4 knowledge graph
  • DB100k+, YAGO3-10+ and NELL-995+, semantically enriched datasets for link prediction
  • Wikidata Thematic Subgraph Selection, datasets to train and evaluate algorithms to select thematic subgraphs of interest in a large knowledge graph from seed entities of interest
  • Wikidata Thematic Subgraph Selection - An Artwork Dataset for experiments to select thematic subgraphs of interest in a large knowledge graph from seed entities of interest
  • Pharmacogenomics datasets for Ontology Matching, datasets based on the PGxLOD knowledge graph to evaluate Ontology Matching algorithms on a structure-based instance matching task from the domain of pharmacogenomics
  • analogical-pruning, an analogy-based pruning model for subgraph selection
  • PyGraft, a Python tool to generate synthetic schemas and knowledge graphs
  • KGPrune, an API and Web application to extract subgraphs from Wikidata around user-input entities of interest while avoiding topical drift
  • Code of other experiments on GitHub
